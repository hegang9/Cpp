首先给出一个类模板的示例：
template<class nameType,class priceType = int>
class Car
{
public:

	Car(nameType name,priceType price)
	{
		this->C_name = name;
		this->C_price = price;
	}
	 
	void printCar()
	{
		cout << "车的名字是：" << C_name << endl;
		cout << "车的价格是：" << C_price << endl;
	}
	 
	nameType C_name;
	priceType C_price;

};

## 类模板和函数模板的区别
1. 类型推导：在C++17之前，类模板无法自动推导出类型，也就是说，在需要使用类声明时，需要使用<>来显式指定类型（例如car<string,int> a）；
2. 默认值：类模板可以指定默认类型，例如上面的类模板，可以使用car<string>，省略第二个类型，使用默认值int
