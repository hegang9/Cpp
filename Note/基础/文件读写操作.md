# 文件读写操作

## 包含头文件

```cpp
#include <fstream>  // 文件读写头文件
```

## 文件写入

### 基本步骤

```cpp
#include <iostream>
#include <fstream>
using namespace std;

int main()
{
    // 创建文件输出对象
    ofstream outFile;
    
    // 打开文件
    outFile.open("Note/文件读写操作.txt");
    
    // 检查文件是否成功打开
    if (outFile.is_open()) {
        // 向文件写入内容
        outFile << "文件读写操作" << endl;
    } else {
        cout << "没有打开文件" << endl;
    }
    
    // 关闭文件
    outFile.close();
    
    return 0;
}
```

## 文件读取

### 基本步骤

```cpp
#include <iostream>
#include <fstream>
#include <string>
using namespace std;

int main()
{
    // 创建文件输入对象
    ifstream inFile;
    
    // 打开文件
    inFile.open("Note/文件读写操作.txt");
    
    // 从文件读取内容
    string in;
    inFile >> in;
    cout << in;
    
    // 关闭文件
    inFile.close();
    
    return 0;
}
```

## 关键类和方法

### ofstream（文件输出流）

- **用途**：向文件写入数据
- **方法**：
  - `open(filename)`：打开文件
  - `is_open()`：检查文件是否成功打开
  - `<<`：输出运算符，用于写入数据
  - `close()`：关闭文件

### ifstream（文件输入流）

- **用途**：从文件读取数据
- **方法**：
  - `open(filename)`：打开文件
  - `is_open()`：检查文件是否成功打开
  - `>>`：输入运算符，用于读取数据
  - `close()`：关闭文件

## 最佳实践

1. 总是检查文件是否成功打开（使用 `is_open()` 方法）
2. 在完成文件操作后，及时关闭文件（使用 `close()` 方法）
3. 合理处理文件打开失败的情况
4. 使用路径时考虑跨平台兼容性
