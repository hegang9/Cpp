# 循环和关系表达式

## 自增自减运算符

### 前缀与后缀的区别

| 特性 | 前缀 | 后缀 |
|-----|------|------|
| 返回值类型 | 左值 | 右值 |
| 返回值特点 | 可以对其进行赋值 | 不可赋值 |
| 执行顺序 | 先自增/自减，后参与运算 | 先参与运算，后自增/自减 |
| 临时对象 | 不生成临时对象 | 需创建临时对象保存旧值 |
| 性能 | 效率较高 | 导致额外构造和析构成本 |

### 与指针的解除引用运算符的优先级

#### 前缀递增、前缀递减和解除引用

- 优先级相同
- 以**从右到左**的方式进行结合

```cpp
*++p  // 等价于 *(++p)
++*p  // 等价于 ++(*p)
```

#### 后缀递增和后缀递减

- 优先级相同，但比前缀运算符的优先级**高**
- 以**从左到右**的方式进行结合

```cpp
*p++  // 等价于 *(p++)，即先对 p 解除引用再自增
```

## 基于范围的 for 循环

### 语法

```cpp
for (int& i : prices) {
    cout << i << " ";
}
```

### 示例

```cpp
int prices[10] = { 1, 2, 3, 4, 5, 6, 7, 8, 9, 0 };
for (int& i : prices) {
    cout << i << " ";
}
cout << endl;
```

### 特点

- 可以遍历数组和容器的所有元素
- 使用引用 `&` 可以修改原数组中的元素
- 使用 const 引用 `const&` 可以避免复制开销并防止修改
