C++三种STL容器：
顺序（序列）容器：vector、list、deque、array、forward_list等
容器	        特点                                  底层数据结构
vector	        动态数组，随机访问快，尾部插入快        动态数组
list	        双向链表，任意位置插入/删除快           双向链表
deque	        双端队列，头尾都能快速插入/删除         分段连续数组，指针数组
array	        固定大小数组（C++11）                  固定大小数组
forward_list	单向链表（C++11），比list省空间         单向链表


关联容器：set、map、multiset、multimap等基于 “红黑树” 的有序存储
容器	        特点
set	            唯一元素自动排序
multiset	    元素允许重复
map	            键值对，键唯一，按键排序
multimap	    键值对，键可重复，按键排序


无序关联容器：unordered_map, unordered_set, unordered_multimap, unordered_multiset等基于 “哈希表” 的无序存储
容器	            特点
unordered_set	    无序唯一集合
unordered_map	    无序键值对
unordered_multiset	无序可重复集合
unordered_multimap	无序可重复键值对

容器适配器：stack、queue、priority_queue等
容器            特点                底层数据结构
stack           后进先出            默认基于deque
queue           先进先出            默认基于deque
priority_queue  优先级高的元素先出   默认基于vector

注：STL 还包括配套的迭代器、算法、仿函数、适配器等组件，红黑树和哈希表的相关知识在DataStruct中
